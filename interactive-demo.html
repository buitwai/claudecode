Install Claude Code on macOS using npm and configure your environment.

Linux Installation

Set up Claude Code on your Linux distribution with package managers.

curl -fsSL https://claude.ai/install.sh | sh
claude --help

Windows Installation

Get started with Claude Code on Windows using PowerShell or WSL.

npm install -g @anthropic-ai/claude-code
claude.exe --version
